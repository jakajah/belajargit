☕ Menu Perintah Git Cafe!

Selamat datang di area ngoding! Ini dia perintah-perintah Git yang sering kamu pakai:

Git Init

Membuat repositori Git baru di folder saat ini.

git init

Git Add

Menambahkan perubahan file ke staging area.

git add .

Git Commit

Menyimpan perubahan ke riwayat repositori.

git commit -m "Pesan commit"

Git Status

Melihat status perubahan file.

git status

Git Log

Melihat riwayat semua commit.

git log

Git Branch

Melihat atau membuat branch baru.

git branch nama-baru

Git Checkout

Berpindah antar branch.

git checkout nama-branch

Git Merge

Menggabungkan perubahan dari branch lain.

git merge nama-branch

Git Pull

Mengambil & menggabungkan dari remote.

git pull origin main

Git Push

Mengirim commit ke remote.

git push origin main

Git Clone

Menyalin repositori dari remote.

git clone <url>

Git Remote

Mengelola koneksi remote.

git remote -v
