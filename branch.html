🌳 Git Branch

Dalam Git, branch (cabang) adalah sebuah alur kerja atau lini pengembangan yang terisolasi. Anggap saja seperti sebuah duplikat dari proyek utama dimana Anda bisa bebas melakukan perubahan tanpa mengganggu pekerjaan di alur utama (biasanya branch `main` atau `master`).

Perintah-Perintah Dasar Branch

Membuat Branch Baru

Perintah ini akan membuat sebuah branch baru, namun Anda masih berada di branch saat ini.

git branch [nama-branch-baru]

Melihat Semua Branch

Untuk melihat daftar semua branch. Branch yang aktif akan ditandai dengan bintang (*).

git branch

Berpindah Branch

Untuk beralih dari branch saat ini ke branch lain.

git checkout [nama-branch-tujuan]

Membuat dan Langsung Berpindah Branch (Shortcut)

Ini adalah cara cepat untuk membuat branch baru dan langsung beralih ke sana.

git checkout -b [nama-branch-baru]

Menggabungkan Branch (Merge)

Menggabungkan perubahan dari branch lain ke branch Anda saat ini.

git merge [nama-branch-sumber]

Menghapus Branch

Menghapus branch lokal yang sudah tidak terpakai.

git branch -d [nama-branch-yang-akan-dihapus]
